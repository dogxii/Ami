### Features

- 记忆你：偏好、重要日子、正在做的事，都被好好珍藏（长期记忆 + 语义检索）
- 温柔对话：统一人格与边界，体贴回应、不越界承诺
- 可插拔模型：自由切换云端/本地 LLM，体验始终如一
- 安全与透明：可导出/删除，数据最小化，隐私有选择

---

### Philosophy

- 温柔，但不过度：Ami 会提醒你休息与自我关怀
- 真实的边界：她始终坦诚自己是 AI
- 尊重你的隐私：你说的话，由你决定去留

---

### Naming

“Ami”在法语里是“朋友”。  
愿她成为你温柔、可靠、常在身边的朋友。
